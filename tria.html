Triadic Colours

Triadic colour schemes consist of three colours spaced evenly on the colour wheel. Hence the name Triadic, meaning "group of three".

Triadic colour schemes are similar to complementary colour schemes in that it’s colours contrast. However triadic colour schemes use three different colour hues, where complementary colour schemes use only two. This allows more freedom and variety in images as more colours can be used.

Triadic colour schemes provide a balance between contrast and harmony. Like complementary schemes, triadic colour schemes are vibrant and contrasting since the colours are split some distance between each other. Though separate, the comparatively tighter spacing of the colours allow this scheme to remain harmonious.

Example: Orange, Green, and Purple
